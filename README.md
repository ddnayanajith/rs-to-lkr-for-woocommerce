# Rs to LKR for WooCommerce

A simple and lightweight WooCommerce plugin that changes the price display from **Rs** to **LKR** for Sri Lankan Rupees.

This plugin only affects how prices are **displayed**. It does **not** convert currency values.

---

## Features

- Changes currency symbol from Rs to LKR
- Clean and readable price format
- Lightweight and fast
- No settings required
- Works automatically when WooCommerce currency is set to LKR

---

## Requirements

- WordPress
- WooCommerce
- WooCommerce currency set to **LKR**

---

## Installation

### Upload via WordPress
1. Download the plugin ZIP from GitHub
2. Go to **WordPress Admin → Plugins → Add New → Upload Plugin**
3. Upload the ZIP file
4. Activate the plugin
